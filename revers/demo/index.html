Project Portal: Revers

A cute little microservice to reverse any string supplied to it. It even works with strings with crazy unicode and emojis! CATS 🐈 becomes 🐈 STAC, for example.

Endpoints

  • GET /revers: shows this very page
  • POST /revers: reverses the request body
  • GET /revers/:string: reverses the string in the url
