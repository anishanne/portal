Project Portal: Heart

This is a simple heartbeat demo page. All it does is send a heartbeat with the script, as well as houses some documentation.
See the other projects »

Endpoints

  • /heart: responds with the current number of sessions
  • /heart/demo: this page
  • /heart/beat: houses a simple script to send a heartbeat
  • /heart/beat/:session: sends a heartbeat

Script

Just include the heartbeat script in every page to automatically send a heartbeat. 

<script src='https://portal.kognise.dev/heart/beat'></script>
